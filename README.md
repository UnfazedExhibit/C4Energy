# Unfazed × C4 Energy — Atlanta Cultural Partnership

A single-page partnership dossier. Fully self-contained: all fonts, images, and styles are inlined into `index.html`, so there are no external dependencies and nothing to break.

## View it live with GitHub Pages

1. Create a new repository and upload the contents of this folder to the **root** (keep `index.html` at the top level).
2. Go to **Settings → Pages**.
3. Under **Source**, choose **Deploy from a branch**, select your `main` branch and the `/ (root)` folder, then **Save**.
4. Your site goes live at `https://<your-username>.github.io/<repo-name>/` within about a minute.

## Files

- `index.html` — the complete, standalone page (open it directly in any browser).
- `.nojekyll` — tells GitHub Pages to serve the files as-is.
- `README.md` — this file.

> Note: In the plain GitHub file view, an HTML file shows as source code. It only renders as a web page through GitHub Pages (steps above).
